무겁고 복잡한 프로그램은 가라, 꼭 필요한 핵심 기능만 담았습니다.

채널 VOD 일괄 파싱

스트리머의 채널 링크만 붙여넣으면 모든 VOD를 자동으로 분석하여 대기열에 차곡차곡 추가합니다.

스마트 구간 자르기

수 시간짜리 영상 전체를 받을 필요 없이, 원하는 하이라이트 시간대만 입력하면 그 부분만 잘라서 저장합니다.

메타데이터 완벽 보존

다운로드된 영상 파일 내부에 치지직 원본 썸네일 이미지와 방송 제목, 스트리머 정보를 영구적으로 박아넣습니다.

스스로 진화하는 자동 업데이트

프로그램 실행 시 최신 버전을 확인하고, 동의 한 번이면 깃허브에서 새 파일을 받아 스스로 덮어쓰고 재시작합니다.

포맷 및 화질 선택

원본 스트림을 손실 없이 추출하여 1080p 60fps 최고 화질로 저장하며, MP4 및 MKV 포맷 변환을 완벽 지원합니다.

내 입맛대로 설정 저장

자주 사용하는 저장 경로, 화질 옵션, 브라우저 쿠키 등의 개인 설정을 기억해두어 켤 때마다 다시 맞출 필요가 없습니다.

자주 묻는 질문 (FAQ)

프로그램 사용 전 꼭 읽어주세요!

실행 시 스마트스크린(보안 경고) 창이 떠요. 바이러스인가요?

파이썬 코드를 윈도우 단일 실행 파일(.exe)로 변환해 주는 PyInstaller의 고질적인 특성상, 윈도우 디펜더(Windows Defender) 등 일부 백신 프로그램이 이를 '알려지지 않은 프로그램(오진)'으로 인식하여 실행을 차단하는 현상입니다.

안심하세요! 본 프로그램은 어떠한 악성코드도 포함하고 있지 않으며, GitHub 리포지토리에 모든 오픈소스 코드가 투명하게 공개되어 있습니다. 차단 창이 뜨면 '추가 정보' → '실행'을 눌러주시거나 백신 예외 항목으로 등록해 주시면 정상 작동합니다.

클립(Clip) 영상은 다운로드할 수 없나요?

현재 치지직 측의 클립 전용 API 구조 변경 및 봇 방어벽(401 Unauthorized 등) 강화로 인해 클립 다운로드는 공식적으로 지원하지 않습니다. 일반 VOD 다시보기 영상 및 스트리머 채널 주소 위주로 사용해 주시기 바랍니다.

쿠키(Cookies) 옵션은 언제 사용하는 건가요?

성인 인증(19+)이 필요한 VOD나 채널 멤버십(구독자) 전용 영상을 다운로드할 때 필요합니다. 평소 치지직을 시청하시는 브라우저(예: Chrome)를 프로그램 내 옵션에서 선택해 두시면, 다운로더가 해당 브라우저의 로그인 상태(쿠키)를 빌려와 안전하게 권한을 우회하여 다운로드합니다.
