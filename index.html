무료 다운로드 (Windows) 기능 둘러보기

Windows 10/11 전용 (무설치 단일 실행 파일)

직관적이고 강력한 기능들

개발자가 직접 쓰려고 만든, 꼭 필요한 핵심 기능만 담았습니다.

채널 전체 일괄 다운로드

스트리머의 채널 링크만 붙여넣으면 모든 VOD를 자동으로 분석하여 대기열에 차곡차곡 추가합니다.

원하는 구간만 자르기 (클립)

수 시간짜리 영상 전체를 받을 필요 없습니다. 원하는 하이라이트 시간대만 입력하면 그 부분만 잘라서 저장합니다.

메타데이터 & 썸네일 자동 삽입

다운로드된 영상 파일 내부에 치지직 원본 썸네일 이미지와 방송 제목, 스트리머 정보를 영구적으로 박아넣습니다.

스마트 자동 업데이트

프로그램 실행 시 최신 버전을 확인하고, 동의 한 번이면 깃허브에서 새 파일을 받아 스스로 덮어쓰고 재시작합니다.

1080p 최고 화질 보장

원본 스트림을 손실 없이 추출하여 1080p 60fps 최고 화질로 저장하며, MP4 및 MKV 포맷 변환을 완벽 지원합니다.

개인 환경 설정 저장

자주 사용하는 저장 경로, 화질 옵션, 브라우저 쿠키 등의 개인 설정을 기억해두어 켤 때마다 다시 맞출 필요가 없습니다.
