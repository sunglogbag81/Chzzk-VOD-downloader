자주 묻는 질문

사용 중 궁금한 점이 있다면 먼저 확인해 보세요.

실행 시 스마트스크린(보안 경고) 창이 뜨거나 백신이 지워요.

파이썬 코드를 윈도우 단일 실행 파일(.exe)로 변환하는 PyInstaller 특성 탓에, 윈도우 디펜더가 '알려지지 않은 프로그램'으로 오진하는 현상입니다. 코드는 GitHub에 모두 투명하게 공개되어 있으니 안심하시고, 차단 창에서 '추가 정보' → '실행'을 누르거나 예외 항목으로 설정해 주세요.

쿠키(Cookies) 우회 옵션은 언제 쓰나요?

성인 인증(19+) 영상이나 채널 구독자 전용 영상을 받을 때 씁니다. 평소 치지직에 로그인해 두는 웹 브라우저를 옵션에서 선택하면, 프로그램이 로그인 세션을 빌려와서 안전하게 영상을 받아옵니다.

짧은 '클립(Clip)' 영상도 다운로드할 수 있나요?

현재 치지직 서버 측의 클립 전용 API 방어벽 강화로 인해 클립 다운로드는 제한되어 있습니다. 일반 VOD(다시보기) 용도로 사용해 주시기 바랍니다.
