POWERFUL FEATURES

개발자가 직접 쓰려고 깎은 기능들

군더더기는 빼고 오직 VOD 보존에 필요한 핵심 기능만 꽉 채웠습니다.

채널 전체 VOD 일괄 파싱

스트리머의 채널 메인 주소만 입력하면 해당 채널에 업로드된 모든 VOD 목록을 한 번에 스캔하여 대기열에 추가합니다.

스마트 구간(시간) 자르기

수 시간짜리 영상 전체를 받을 필요가 없습니다. 원하는 하이라이트의 시작 및 종료 시간만 입력해 필요한 부분만 깔끔하게 저장하세요.

메타데이터 완벽 내장

단순히 영상 파일만 덩그러니 남지 않습니다. 치지직의 원본 썸네일, 영상 제목, 스트리머 채널명 등의 메타데이터를 파일에 영구적으로 삽입합니다.

세밀한 화질 및 포맷 제어

1080p 최고 화질부터 데이터 절약을 위한 저화질, 그리고 영상 없이 오디오(MP3)만 추출하는 기능까지 지원합니다. MP4/MKV 병합도 자유롭습니다.

성인/구독자 전용 우회 (쿠키)

크롬, 엣지 등 평소 사용하는 브라우저의 로그인 상태(쿠키)를 불러와 성인 인증이 필요하거나 멤버십 전용인 영상도 막힘없이 다운로드합니다.

스스로 갱신하는 자동 업데이트

매번 새 버전을 깃허브에서 찾아 다운로드할 필요가 없습니다. 앱 실행 시 새 버전을 감지하면 원클릭으로 덮어쓰고 자동 재시작합니다.

QUESTIONS & ANSWERS

자주 묻는 질문

사용 중 막히는 부분이 있다면 먼저 확인해 보세요.

실행 시 스마트스크린(보안 경고) 창이 뜨거나 백신이 지워요.

파이썬 코드를 윈도우 단일 실행 파일(.exe)로 변환해 주는 PyInstaller의 고질적인 특성 탓에, 윈도우 디펜더가 이를 '알려지지 않은 프로그램'으로 오진하는 현상입니다.

코드 전체가 GitHub에 공개되어 있는 투명한 오픈소스이므로 안심하셔도 됩니다. 차단 창에서 '추가 정보' → '실행'을 누르거나 백신 예외 항목으로 설정해 주세요.

PC에 파이썬(Python)이 깔려 있어야 하나요?

아니요, 전혀 필요 없습니다. 프로그램 실행에 필요한 모든 파이썬 환경과 FFMPEG 라이브러리가 .exe 파일 하나에 압축되어 포함되어 있습니다. 다운로드 후 즉시 실행만 하시면 됩니다.

쿠키(Cookies) 우회 옵션은 언제 쓰나요?

성인 인증(19+) 영상이나 채널 구독자(멤버십) 전용 VOD를 다운받을 때 사용합니다. 평소 치지직에 로그인해 두는 브라우저(예: 크롬, 엣지)를 옵션에서 선택하면, 프로그램이 로그인 권한을 잠시 빌려와서 안전하게 영상을 추출합니다.

짧은 '클립(Clip)' 영상도 다운로드할 수 있나요?

현재 치지직 서버 측의 클립 전용 API 구조 변경 및 봇 방어벽(401 Unauthorized 등) 강화로 인해 클립 다운로드는 제한되어 있습니다. 다시보기(VOD)와 스트리머 메인 채널 주소 용도로만 사용해 주세요.
